**About the Project**--
This project is a sales analysis dashboard created in Power BI using Blinkit-style retail data.
The main goal of this dashboard is to understand sales performance across different outlets, item categories, and locations, and to present the insights in a simple and interactive way.

**Business Questions Covered**--
1-Which outlet location (Tier 1, Tier 2, Tier 3) generates the highest sales?

2-Which product categories contribute most to overall revenue?

3-Does outlet size have an impact on total sales?

3-How do average sales differ across outlet types?

4-Which outlet type sells the highest number of items?

5-What is the sales split between Low Fat and Regular products?

6-How have sales changed based on outlet establishment year?

7-Which supermarket format performs better in terms of sales?

8-How do customer ratings vary across outlet types?

**Key Insights**
.Tier 3 outlets contribute the highest share of total sales.
.Low Fat items generate more revenue compared to Regular items.
.Supermarket Type 1 shows the strongest overall sales performance.
.Medium and High outlet sizes perform better than small outlets.

**Dashboard Features**
KPI cards for quick performance overview
Interactive filters for outlet size, location, and item type
Trend analysis using line charts
Category-wise and outlet-wise sales breakdown

**Conclusion**
This dashboard helps in quickly identifying top-performing outlets and products and supports data-driven decision making for retail businesses.
